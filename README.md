# autodrive


### 📌 브랜치 전략

- `main`: 운영 배포용
- `feature/{기능이름}`: 기능 추가
- `fix/{버그설명}`: 버그 수정
- `refactor/{리팩토링내용}`: 리팩토링 작업

### 🔖 커밋 태그

| 태그 | 설명 |
|------|------|
| ✨ Feat | 기능 추가 |
| 🐛 Fix | 버그 수정 |
| ♻️ Refactor | 리팩토링 |
| 📄 Docs | 문서 작업 |
| 🔧 Chore | 설정 변경 |
| ✅ Test | 테스트 코드 |
| 🗃️ Rename | 파일/폴더명 변경 |
| 🚀 Deploy | 배포 관련 |
